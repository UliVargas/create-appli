# Create App

Create projects with templates.

## Usage

```bash
# Interactive mode
npx create-appli

# Override template
npx create-appli --template <template>

# Override name
npx create-appli --name <name>

# Get Command help
npx create-appli --help
```

## Installation

```bash
npm install -g create-appli
```
