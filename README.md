# Create App

Create projects with templates.

## Usage

```bash
# Interactive mode
npx create-app

# Override template
npx create-app --template <template>

# Override name
npx create-app --name <name>

# Get Command help
npx create-app --help
```

## Installation

```bash
npm install -g create-app
```
